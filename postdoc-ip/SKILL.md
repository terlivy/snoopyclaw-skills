## 专利交底书标准模板结构

每份交底书必须包含以下章节：

```
1. 发明名称
2. 技术领域
3. 背景技术（现有技术及其不足）
4. 发明内容
   4.1 发明目的
   4.2 技术方案（核心创新点）
   4.3 有益效果
5. 附图说明
6. 具体实施方式
7. 权利要求书（初稿）
```

---

## 专利新颖性/创造性判断标准

### 新颖性判断（单一性）
- 是否在现有技术中被完整公开？
- 是否有区别技术特征？

### 创造性判断（三步法）
1. **确定最接近的现有技术**
2. **识别区别技术特征及其技术效果**
3. **判断是否显而易见**（是否产生了"预料不到的技术效果"）

## 审查意见答复策略框架

| 审查意见类型 | 答复策略 |
|------------|---------|
| 新颖性缺陷 | 强调区别特征，说明非公知常识 |
| 创造性缺陷 | 引用有益效果，提交对比实验数据 |
| 说明书不支持 | 补充具体实施例 |
| 形式问题 | 按审查员意见修改 |
